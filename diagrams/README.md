# Diagrams

Mermaid source files (`.mmd`) are the single source of truth.  
PNG exports are generated from these and committed alongside.

| File | Description |
|---|---|
| `system-architecture.mmd` | Full system overview — VaultPay + AuthShield |
| `database-erd.mmd` | Entity Relationship Diagram for all 8 tables |
| `send-money-flow.mmd` | Send Money 8-step security flow |
| `auth-flow.mmd` | JWT dual-mode authentication flow |

> Diagrams are created in **Phase 5** of the repo build.
